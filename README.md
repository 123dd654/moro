# MORO
More Open, Real, Ours

## 01. 기획의도, 목표, 타겟
### [기획 의도]
세무사 사무실 근무 같은 직종 지인들과 오픈채팅방에서 질문하고 소통했는데, 전용 플랫폼 없이 네이버카페나 오픈채팅방에 의존해야 하는 불편함을 느꼈습니다.<br/>
또한 수작업으로 급여를 역계산하던 경험에서 역계산기 아이디어를 떠올려 직접 만들게 됐습니다.

### [목표]
* 누구나 쉽게 사용할 수 있는 세금 계산기 제공
* 세무 관련 종사자들이 자유롭게 질문하고 소통할 수 있는 공간 마련

### [타겟]
* 세무사 사무실 종사자
* 급여 계산이 필요한 직장인
* 세금 정보가 필요한 일반인

## 02. 기술 스택

| 분류 | 기술 |
|------|------|
| Frontend | React, Vite, JavaScript |
| 상태관리 | React Hooks (useState) |
| Database | Firebase Firestore |
| 배포 | Firebase Hosting |
| API | 국세청 사업자등록조회 Open API |


## 03. 주요 기능

- 급여 순계산기
- 주휴수당 계산기
- 급여 역계산기
- 사업자등록 조회

## 04. 개발 예정 기능

- 로그인, 회원가입
- 세금 관련 질문 게시판
- 사용자 간 소통 커뮤니티
- 세무 일정 확인
